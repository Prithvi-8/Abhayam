# ABHAYAM – Charitable Society Website

Official website of ABHAYAM Charitable Society, serving the community since 2012.

This project provides an online presence for Abhayam, showcasing its mission, programs, and activities, and allowing users to contact the organization through a web form.

---

## Features
- Responsive static website
- Information about Abhayam and its programs
- Contact form with email notification
- Donation QR modal
- Integrated chatbot for basic assistance

---
